# Workflows

Markdown SOPs that define how to accomplish recurring tasks using the WAT framework.

## Structure of a workflow file

```
# Workflow Name

## Objective
What this workflow accomplishes.

## Inputs
- input_1: description
- input_2: description

## Tools Used
- tools/script_name.py

## Steps
1. Step one
2. Step two

## Expected Output
What success looks like.

## Edge Cases & Notes
- Known rate limits, quirks, or gotchas
```
